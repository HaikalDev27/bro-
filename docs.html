About GlowText

GlowText is created by FumetsuChigura member (Haikal Apriliano) with html, css, and javascript language. This site is hosted with github (haikaldev27.github.io/GlowTxt). GlowText is already verified by google search console, this site is inspired by some tutorial in youtube.

GlowText is used to make your text shine and cool, you can use it as content for your social media or you can use it whatever you want, you can also request feature to dev for next update. If you don't know how to use this site, you can edit the text by clicking the "hello" text and you can change the color by clicking the gear icon and choosing a color
